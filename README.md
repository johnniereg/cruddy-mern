# I Fed A Duck

An application that allows a scientist to crowdsource info about public duck feeding.

## See the [app live](http://www.ifedaduck.club/)

### Dependencies

* axios
* babel-cli
* babel-preset-env
* babel-preset-es2015
* body-parser
* dotenv
* ejs
* express
* mongoose
* nodemon
* react
* react-bootstrap
* react-dom
* react-modal
* react-router-dom
* react-scripts

### Getting Started

Install dependencies.
Create a .env file based off of the example.
Run database migrations and seed.
Run dev script to compile javascript and start server.

```
npm install
npm run start
open http://localhost:3000
```
